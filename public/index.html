Proyecto personal inspirado en la experiencia en Alvaco. Aplicacion .NET MAUI con API REST propia en Azure y base de datos Azure SQL. Los usuarios se registran y dan de alta a sus mascotas; cada mascota genera un QR unico que al escanearse abre una pagina web con los datos del perro y del dueno, con boton para enviar la ubicacion al dueno via WhatsApp. Incluye integracion con Google Maps para que veterinarias registren su clinica y los usuarios encuentren atencion de emergencia cercana.

C# | .NET MAUI | ASP.NET Core | Azure SQL | Azure App Service | Entity Framework | Google Maps API | QR | WhatsApp API
